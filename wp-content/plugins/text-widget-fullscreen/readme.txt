This is a WordPress plugin that will add a fullscreen button inside your Text Widget.


== Description ==


This is a WordPress plugin that will add a fullscreen button functionality to your Text Widget editor.

After you press the fullscreen button the editor will take all free space of your screen.

Now you can view and inspect your text easier.



== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. That's it!
